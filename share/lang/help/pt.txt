USO:
  aura [módulo]                    Executar módulo (modo terminal)
  aura waybar [módulo]             Executar módulo (modo Waybar JSON)
  aura action [módulo] [ação]      Executar ação do módulo
  aura modules [comando]           Gerenciar módulos
  aura cache [comando]             Gerir cache

MODOS DE EXECUÇÃO:
  terminal                          Saída simples no console
  waybar                            Saída JSON para Waybar
  action                            Executar ações de módulos

GESTÃO DE MÓDULOS:
  modules list                      Listar todos os módulos
  modules info [módulo]             Mostrar informação detalhada

GESTÃO DE CACHE:
  cache info                        Mostrar estado de toda a cache
  cache info [chave]                Mostrar estado de uma entrada
  cache clear [chave]               Limpar entrada específica
  cache clear-all                   Limpar toda a cache

COMANDOS GERAIS:
  help                              Mostrar esta ajuda
  version                           Mostrar versão
  list                              Atalho para 'modules list'

EXEMPLOS:
  aura power_info                   Executar módulo power_info
  aura waybar power_info            Power info em formato Waybar
  aura action radio toggle          Alternar reprodução da radio
  aura action weather refresh       Forçar atualização do tempo
  aura modules info weather         Ver informação do módulo
  aura cache info                   Ver estado da cache
  aura cache clear weather          Limpar cache do tempo
  aura list                         Listar todos os módulos

VARIÁVEIS DE AMBIENTE:
  AURA_LANG                         Idioma (auto|en|es|pt, padrão: auto)
  AURA_DEBUG                        Ativar modo debug (true|false)

CONFIGURAÇÃO:
  ~/.config/auraconfig/.env         Arquivo de configuração
  ~/.local/share/auraconfig/        Módulos de usuário

CAMINHOS:
  ~/.local/bin/aura                 Executável
  ~/.local/lib/auraconfig/          Bibliotecas
  ~/.local/share/auraconfig/        Recursos
  ~/.config/auraconfig/             Configuração
  ~/.cache/auraconfig/              Cache
